# Shark Game 🦈

シンプルで楽しいブラウザゲーム。サメを操作して魚を捕まえましょう！

## 遊び方 🎮

1. 矢印キーでサメを操作
2. できるだけ多くの魚を捕まえる
3. 制限時間は30秒
4. ハイスコアに挑戦！

## 技術スタック 🛠

- HTML5
- CSS3
- JavaScript
- Phaser 3 (ゲームエンジン)

## 開発環境のセットアップ 💻

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm start
```

## プレイ方法 🎯

1. ブラウザで `http://localhost:8080` を開く
2. 矢印キーでサメを操作
3. 魚を捕まえてスコアを稼ぐ
4. 30秒以内にハイスコアを目指す！
