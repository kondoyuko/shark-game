# Shark Game

シンプルなブラウザゲーム。サメを操作して魚を捕まえるゲームです。

## 遊び方

1. 矢印キーでサメを操作
2. できるだけ多くの魚を捕まえる
3. 制限時間は30秒
4. ハイスコアに挑戦！

## 開発環境のセットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm start
```

## GitHub Pagesでの公開方法

1. GitHubでリポジトリを作成
2. Settings → Pages で Source を `main` に設定
3. Save をクリック

その後、`https://ユーザー名.github.io/shark-game/` でアクセス可能になります。
